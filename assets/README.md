# Task Manager

Pequeño gestor de tareas desarrollado con:

- PHP
- JavaScript Vanilla
- AJAX (Fetch API)
- Bootstrap

## Funcionalidades

✔ Crear tareas
✔ Listar tareas
✔ Marcar como completadas
✔ Actualización dinámica sin recargar la página
✔ Persistencia usando archivo JSON

## Estructura

index.php → interfaz principal  
store.php → guarda tareas  
update.php → actualiza estado  
tasks.json → almacenamiento  

## Cómo ejecutar

1. Clonar repositorio
2. Ejecutar en servidor local (XAMPP / WAMP)
3. Abrir:

http://localhost/task-manager
